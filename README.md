# Password Checker

A simple Python program I made while learning Python. 
It checks how strong your password is and gives you advice on how to improve it.

## How it works

You type in your password and it checks for:
- Length (16+ characters)
- Uppercase letters
- Lowercase letters
- Numbers
- Special characters

It gives your password a score out of 5 and tells you what's missing.

## How to run it

Make sure you have Python installed, then run it one of two ways:

**Interactive mode:**
python password-checker.py

**Command-line mode:**
python password-checker.py yourpassword
